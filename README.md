## Installation

```bash
$ npm install
```

## Configuration
The following environment variable should be set in your `.env` file
```
PAGE_TITLE=
PROFILE_URL=
PORT=3000
DATABASE_HOST=localhost
DATABASE_PORT=5432
USER_API_TOKEN=
ADMIN_API_TOKEN=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
